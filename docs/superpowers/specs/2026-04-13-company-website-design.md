## 技术方案

### 文件改动

| 文件 | 操作 |
|------|------|
| `src/app/page.tsx` | 替换 redirect 为官网首页组件 |
| `src/app/layout.tsx` | 修改 title 和 description 元数据为公司名称和描述 |
| `src/app/globals.css` | 添加 `html { scroll-behavior: smooth }` |

### 元数据更新（layout.tsx）

```ts
title: "平顶山市帮颂商贸有限公司"
description: "平顶山市帮颂商贸有限公司 - 专注电子产品与人工智能硬件销售，为您提供优质的产品与服务"
```

### 架构要点

- 官网为服务端组件（Server Component），不依赖认证
- 管理后台路由 `/admin/*` 不受影响，middleware 仅保护 admin 路由
- 不新增 API 路由，纯静态页面
- 复用现有 Tailwind CSS 4 + Lucide React 图标
- 原根路径的 `redirect("/admin/login")` 被移除，改为展示官网页面
- 访问 `/admin` 仍由 middleware 保护，未登录会重定向到 `/admin/login`，此流程不变

## 验证方式

1. `npm run dev` 启动开发服务器
2. 访问 `/` 应展示官网首页（不再重定向）
3. 访问 `/admin` 应正常跳转到管理后台登录页（middleware 拦截）
4. 官网各区块锚点导航正常工作，滚动平滑
5. 手机端响应式布局正常
6. 电话卡片在手机端点击可触发拨号
7. 页脚 ICP/公安备案链接可点击跳转
